# Todo List App

A responsive task management application that allows users to add, edit, delete, and manage daily tasks efficiently.

## Live Demo
https://durga-ranakoti-carry.github.io/Todo-List-App

## GitHub Repository
https://github.com/durga-ranakoti-carry/Todo-List-App

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Local Storage support
- Responsive design
- Clean user interface

- JavaScript
- Local Storage
