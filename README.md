# 📝 To-Do List App

A modern and responsive To-Do List application built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Edit tasks by double-clicking
* Delete tasks
* Data saved using Local Storage
* Responsive design
* Smooth animations and hover effects
* Glassmorphism UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 📸 Preview

This application allows users to manage daily tasks efficiently with a clean and modern user interface.

## 🎯 How to Use

1. Enter a task in the input field.
2. Click the **Add** button or press **Enter**.
3. Double-click a task to edit it.
4. Use the checkbox to mark a task as completed.
5. Click **Delete** to remove a task.

## 💾 Local Storage

Tasks are automatically saved in the browser's Local Storage, so they remain available even after refreshing the page.
